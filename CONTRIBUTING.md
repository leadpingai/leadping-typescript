# Contributing

Thanks for helping improve the Leadping SDK.

## Before You Start

- Search existing issues and pull requests before opening a new one.
- Use the bug, documentation, or feature issue templates when they fit.
- Report security vulnerabilities privately by following `SECURITY.md`.

## Pull Requests

- Keep changes focused and easy to review.
- Include a clear description of the problem and solution.
- Update README or examples when behavior changes.
- Avoid unrelated formatting or generated-file churn.

## Generated SDK Code

This SDK is generated from the Leadping OpenAPI specification. For API shape changes, update the OpenAPI source first and regenerate the SDK instead of hand-editing generated client files.
