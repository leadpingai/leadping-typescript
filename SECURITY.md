# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Email security reports to support@leadping.ai with the subject `Security Report`. Include a description of the issue, affected versions or endpoints, reproduction steps, and any relevant proof of concept.

We will acknowledge reports as quickly as possible and coordinate fixes before public disclosure.

## Supported Versions

Security fixes are provided for the latest published version of this SDK.
